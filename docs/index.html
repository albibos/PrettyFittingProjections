    Introduction

    In these docs, we will be showing you how to use the services Accel provides. To start with, we have CDN loading:

    CDN Loading

    There are 3 main ways to use CDNs to load games. The first being via an iframe on the page, the second via url queries, and the third via localstorage. We will be going over each of these in detail.

    First and foremost, here are some good CDNs to use:

            
    https://raw.githack.com/
    https://gh.maple3142.workers.dev/
    https://rawgitt.glitch.me/
    You can also selfhost a CDN yourself
    by using the repo:
    https://github.com/albibos/rawgit2017!

    Same Page iFrame

    The same page iframe is quite self explanatory. Typically, this iframe is located on a popup or at the bottom of the screen. When the user presses a button, the iframe will be made visible with the src set to the cdn.

    In the above example, we use a dialog element (the easiest way to make a popup) to create the popup. Then, using a function, use the dialog.showModal() to show the popup, and set the src using the functions parameter. Here is the JS: 

            
 function openFrame(id) {
   dialog = document.getElementById("frame-dialog");
   frame = document.getElementById("frame");
   if (dialog.open) {
     dialog.close();
   } else {
     dialog.showModal();
     frame.src = 'https://raw.githack.com/WhoIsEv/relivedassets/main/' + id + '/index.html'; // you can use any cdn, this is just an example!!
   }
 }

 function closedi() {
   dialog = document.getElementById("frame-dialog");
   dialog.close();
 }

    Please refer to the codepen example for anything you need.

    URL Queries

    A url query is a part of a url that assigns a value to a parameter. They are almost always a ?parameter=value. For example, if you wanted to load a game from a url, you would use ?game=game-id. We will be making a games.html example and loading the src in the iframe based on the query.

    Here we have a simple iFrame: 

     
  <iframe id="gameFrame" width="100%" height="600px" frameborder="0"></iframe>

    And its corresponding js: 

            
 const gameId = new URLSearchParams(window.location.search).get('game');

 // Set the iframe src based on the 'id' parameter
 if (gameId) {
     document.getElementById('gameFrame').src = `https://raw.githack.com/WhoIsEv/relivedassets/main/${gameId}/index.html`;
 } else {
     console.error('Missing game parameter.');
 }

    In the above js code we: get the game id parameter from the url and form a proper url for our src via it. If the game id is not found, we log an error to the console. So, if the url is https://example.game/whatever.html?game=2048, our iFrame will display the game 2048. So now, when you redirect to your iframe page with a query of ?game=id, it will load the cdn url with the game!
